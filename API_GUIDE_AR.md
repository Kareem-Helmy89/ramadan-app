# دليل الحصول على واجهات برمجة تطبيقات توليد الصور

## ⚠️ ملاحظة مهمة عن Gemini Nano

**Gemini Nano** من Google **ليس** مناسب لتوليد الصور. هو نموذج موجه للنصوص والمحادثة ويعمل على الأجهزة المحلية.

## أفضل الخيارات لتوليد الصور:

### 1. **Replicate** (الأسهل والأرخص) ⭐ موصى به

**المميزات:**
- سهل الاستخدام
- أسعار معقولة (من $0.002 للصورة)
- يدعم Stable Diffusion و Flux و نماذج أخرى
- لا يحتاج بطاقة ائتمان للبدء (يوجد رصيد مجاني)

**كيفية الحصول على API Key:**
1. اذهب إلى: https://replicate.com
2. سجل حساب جديد (Sign Up)
3. اذهب إلى: https://replicate.com/account/api-tokens
4. انسخ الـ API Token
5. ضعه في ملف `.env` كالتالي:
   ```
   REPLICATE_API_TOKEN=your_token_here
   ```

---

### 2. **OpenAI DALL-E** (الأجود والأغلى)

**المميزات:**
- جودة عالية جداً
- سريع
- موثوق

**العيوب:**
- غالي نسبياً ($0.04 للصورة 512x512)
- يحتاج بطاقة ائتمان

**كيفية الحصول على API Key:**
1. اذهب إلى: https://platform.openai.com
2. سجل حساب جديد
3. اذهب إلى: https://platform.openai.com/api-keys
4. اضغط "Create new secret key"
5. انسخ الـ API Key
6. ضعه في ملف `.env`:
   ```
   OPENAI_API_KEY=sk-your_key_here
   ```

---

### 3. **Hugging Face** (مجاني محدود)

**المميزات:**
- مجاني للاستخدام المحدود
- نماذج متنوعة

**العيوب:**
- أبطأ من الخيارات الأخرى
- محدود في الاستخدام المجاني

**كيفية الحصول على API Key:**
1. اذهب إلى: https://huggingface.co
2. سجل حساب جديد
3. اذهب إلى: https://huggingface.co/settings/tokens
4. أنشئ token جديد
5. ضعه في ملف `.env`:
   ```
   HUGGINGFACE_API_KEY=your_token_here
   ```

---

### 4. **Google Imagen** (إن كان متاحاً)

**ملاحظة:** Imagen من Google قد لا يكون متاحاً للجميع حالياً. تحقق من: https://cloud.google.com/vertex-ai

## التوصية:

**ابدأ بـ Replicate** لأنه:
- سهل الاستخدام
- رخيص
- لا يحتاج بطاقة ائتمان للبدء
- جودة جيدة

---

## الخطوات التالية:

بعد الحصول على API Key:
1. افتح ملف `.env` في مجلد المشروع
2. أضف الـ API Key
3. افتح `app.py`
4. استخدم أحد الأمثلة المرفقة في الملف
